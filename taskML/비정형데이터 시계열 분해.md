
시간정보가 있는 각각의 수학적 또는 상태적 정보의 데이터가 nested array 인 상태일 때, 서로 관련이 있는 correlation 을 조사하면서 어떻게 하면 관련성이 있는 관계를 좀 더 seasonality 를 찾거나 특이한 시점을 detect 하고 predict 할 수 있을까

## 1. 시계열 분해 및 계절성(Seasonality) 파악

데이터에 내재된 반복 패턴을 찾기 위해서는 먼저 성분을 분해해야 합니다.


- STL Decomposition: 데이터를 트렌드(Trend), 계절성(Seasonality), 잔차(Remainder)로 분리합니다. 잔차 부분에서 특이 시점을 찾기가 훨씬 수월해집니다.
    
- Fourier Transform (FFT): 데이터가 복잡한 파동 형태라면, 푸리에 변환을 통해 주파수 영역으로 변환하여 어떤 주기(Daily, Weekly 등)가 가장 강한지 수학적으로 찾아낼 수 있습니다.
    
- Autocorrelation (ACF) & PACF: 자기상관 함수를 통해 과거의 자기 자신이 현재에 얼마나 영향을 주는지 시차(Lag)별로 확인하세요.
    

---

## 2. 특이 시점 탐지 (Anomaly Detection)

인과관계를 알고 계시므로, '원인' 변수의 변화 없이 '결과' 변수가 튀는 지점을 찾는 것이 핵심입니다.

- Isolation Forest / LOF: 고차원 데이터(Flatten 된 데이터)에서 주변 데이터와 동떨어진 고립된 샘플을 효율적으로 찾아냅니다.
    
- Change Point Detection (CPD): 시계열의 통계적 특성(평균, 분산)이 급격히 변하는 지점을 감지합니다. ruptures 같은 라이브러리가 유용합니다.
    
- Dynamic Time Warping (DTW): 서로 다른 두 시계열(원인과 결과)의 속도가 달라도 패턴의 유사성을 측정할 수 있습니다. 특정 이벤트가 결과에 반영되기까지의 Time Lag을 계산할 때 효과적입니다.
    

---

## 3. 다변량 시계열 예측 (Prediction)

단순한 수치를 넘어 비정형적 특성을 반영한 예측 모델이 필요합니다.

- VAR (Vector Autoregression): 여러 시계열 변수가 서로 영향을 주고받는 상황에서 인과관계를 반영한 고전적이고 강력한 통계 모델입니다.
    
- LSTM / GRU: 딥러닝 기반의 순환 신경망으로, Flatten 된 Nested Array의 복잡한 비선형 관계와 긴 시간적 의존성을 학습하기에 적합합니다.
    
- N-BEATS / NHITS: 최근 각광받는 모델로, 시계열을 다항식과 하모닉 성분으로 분해하여 예측하므로 '해석 가능성'과 '계절성' 포착 능력이 뛰어납니다.
    

---

## 4. 인과성 기반의 Feature Engineering 전략

도메인 지식을 활용해 모델의 성능을 극대화하는 방법입니다.

1. Lagged Features: 원인 데이터($X$)가 결과($Y$)에 영향을 주기까지 걸리는 시간을 고려하여 $X_{t-1}, X_{t-k}$ 등의 시차 변수를 생성하세요.
    
2. Rolling Statistics: 일정 시간 창(Window) 안에서의 평균, 표준편차, 최댓값 등을 변수로 추가하면 국소적인 특이 시점을 잡는 데 도움이 됩니다.
    
3. Embedding: 비정형 데이터의 상태 정보(Stateful info)가 범주형이라면, 단순 수치화보다는 Embedding 레이어를 통해 벡터화하여 모델에 주입하는 것이 정보 손실을 줄이는 길입니다.
    

---

### 추천하는 분석 프로세스

1. 상관분석: 교차 상관 계수(Cross-correlation)를 통해 원인과 결과 사이의 최적의 시차(Lag)를 결정합니다.
    
2. 프로펫(Prophet) 활용: Meta에서 만든 이 라이브러리는 휴일 효과, 계절성, 트렌드 변화점을 자동으로 잡아주므로 초기 분석용으로 매우 훌륭합니다.
    
3. 이상치 라벨링: 탐지된 특이 시점을 기반으로 해당 시점의 원인 변수 상태를 역추적(Root Cause Analysis) 합니다.
